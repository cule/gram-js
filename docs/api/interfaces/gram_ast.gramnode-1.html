Interface GramNode

A GramNode is a path composed of two empty paths, which is equivalent to having no children.

  • denoted with matching square brackets: ()
  • path equivalence: [n] =~ [n [] []] =~ (n)
  • identity: yes
  • children: 0
  • labels: yes
  • record: yes
  • path length: 0
  • path cardinality: 1
  • information role: an entity or a noun
