Package gram-parse

Gram Parse

From text literal (a)-->(b) to a data graph syntax tree.

Provides:

  • gram.ne: a grammer written in Nearley
  • gram-parser-plugin: a plugin for the unified text processing framework
  • toAST: a convenience function for parsing text to AST
